# Variables

## What is a Variable?

A variable is a container that stores a value.

Example:

```python
name = "Harshit"
age = 17
```

## Assignment Operator

`=` assigns a value.

Example:

```python
marks = 95
```

## Common Mistakes

❌ Using `==` instead of `=`

Correct:

```python
marks = 95
```

Wrong:

```python
marks == 95
```
